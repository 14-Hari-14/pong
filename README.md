# Title: Pong

## Overview
A ping pong game made with python's pygame library

## Features
Very fun(for the first 5 mins)

## Installation
  
  ### Clone the Repository: 
  git clone https://github.com/14-Hari-14/pong
  cd pong

  ### Run the Python file:
  python main.py
